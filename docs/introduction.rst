Introduction
============

WinePredict is a library designed to facilitate wine price prediction using machine learning models. This documentation provides an overview of the library's features, installation instructions, usage examples, and API reference.
docs/installation.rst
Installation
============

To install the WinePredict library from GitHub, use the following command:

.. code-block:: bash

    pip install git+https://github.com/chris-lawrence-cd/winepredict.git
